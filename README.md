### Login base on React+Flask

## How to run code？

### Create venv

```shell
python3 -m venv venv
source venv/bin/activate
```



### backend

#### prepare

``` shell
cd server
pip3 install -r requirements.txt
```

### Frontend

``` shell
cd client
npm install react-router-dom
npm start
```
